
                                          => Static <=

=> Static keyword is mainly used for memory management.
=> Static keyword can be used in Methods,Variables.
=> Static Variables and methods are based on class rather than the instance of a class.
=> Value to Static Variable or Method is assigned while the class structure is loaded.
=> Static Variables values can be changed only by static methods.
=> Static method can't access non-static Variables.

                                          => This <=

=> This keyword is an reference variable which points to the current object state and variable.
